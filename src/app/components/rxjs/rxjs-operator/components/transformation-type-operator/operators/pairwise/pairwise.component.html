What is the Pairwise Operator?

The Pairwise operator in RxJS is a transformation operator that groups consecutive pairs of values emitted by the source observable and emits them as an array. It is useful when you need to compare or process the current value with the previous value.

Marble Diagram Image Link

Link

Common Use Cases:

  • Tracking Changes: Compare the current value with the previous value (e.g., stock prices, sensor readings).
  • Sliding Window Operations: Perform operations on a sliding window of two consecutive values.
  • Delta Calculations: Calculate the difference or ratio between consecutive values.

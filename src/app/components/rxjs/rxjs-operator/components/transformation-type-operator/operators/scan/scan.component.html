What is the Scan Operator?

The Scan operator in RxJS is a transformation operator that is used to accumulate values over time. It applies an accumulator function to each value emitted by the source observable, passing the result of the accumulator function forward to the next invocation. This makes it similar to the Array.prototype.reduce method in JavaScript, but for observables. It's like reduce, but Emits the current state after each update, unlike reduce, which only emits the final result.

Common Use Cases:

  • Running Totals: Calculate running totals or cumulative values (e.g., sum, average).
  • State Management: Maintain and update a state over time (e.g., a shopping cart).
  • Progressive Data Aggregation: Aggregate data incrementally as new values arrive.

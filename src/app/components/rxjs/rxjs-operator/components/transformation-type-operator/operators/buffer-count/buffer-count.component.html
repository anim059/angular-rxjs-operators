What is the Buffer Count Operator?

The Buffer Count operator in RxJS is another powerful buffering operator. Unlike buffer, which relies on a closing notifier observable to determine when to emit buffered values, bufferCount collects a specified number of values from the source observable and then emits them as an array. Itâ€™s like creating "chunks" or "windows" of values.

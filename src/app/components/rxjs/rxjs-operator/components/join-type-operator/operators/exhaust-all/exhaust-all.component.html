What is the ExhaustAll Operator?

The ExhaustAll operator in RxJS (also known as exhaustMap in some versions) is a flattening operator that is used to handle higher-order observables (observables that emit other observables). It subscribes to the first inner observable emitted by the higher-order observable and ignores all subsequent inner observables until the first one completes. This is useful for scenarios where you want to ignore new emissions while the current one is still active. It is typically used after a higher-order observable and when you have a dynamic number of observables

Common Use Cases:

  • Task Throttling: Ensure that only one task is executed at a time, ignoring new tasks while the current one is active.
  • Event Handling: Ignore new events (e.g., button clicks) while processing the current event.
  • Resource Management: Prevent overlapping operations on shared resources.
